# 🚀 LEO First Subsonic Rocket (Version 1)

> My first rocket design created in OpenRocket as part of my journey
> into aerospace engineering.

## Overview

**LEO First Subsonic Rocket V1** is my first rocket design project. It
was created in OpenRocket to learn rocket design, aerodynamics,
stability, and flight simulation.

## Specifications

  Parameter          Value
  ------------------ ---------------------------
  Project Name       LEO First Subsonic Rocket
  Version            1.0
  Flight Regime      Subsonic
  Rocket Length      700 mm
  Body Diameter      59 mm
  Nose Cone Length   400 mm
  Body Tube Length   300 mm
  Design Software    OpenRocket

## Estimated Performance

-   Maximum altitude: **900--1500 m**
-   Maximum speed: **180--260 m/s**
-   Burn time: **\~3 s**
-   Flight regime: **Subsonic**
